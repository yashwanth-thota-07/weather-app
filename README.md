# 🌤️ TempFinder

TempFinder is a simple and responsive weather web app that shows real-time weather information for any city using the WeatherAPI.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Current temperature display
- 🤒 RealFeel temperature
- ☀️ Weather condition
- 🔥 Heat Index
- 💨 Wind speed
- 🌪️ Wind Gusts

## 🛠️ Built With

- HTML
- CSS
- JavaScript
- WeatherAPI
- Font Awesome

## ⚡ How It Works

1. Enter a city name
2. Click the Search button or press Enter
3. Weather data is fetched from WeatherAPI
4. Weather details are displayed instantly
